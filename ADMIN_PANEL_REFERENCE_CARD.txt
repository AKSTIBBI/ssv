## 🆘 Troubleshooting Quick Fixes

| Problem | Fix |
|---------|-----|
| **404 page not found** | Check full path: `/Project_SSV_Website/real/php/...` |
| **Login fails** | Verify credentials in config.php |
| **Data not saving** | Check dir permissions: `chmod 755 real/json/` |
| **Page slow** | Clear browser cache (Ctrl+Shift+Del) |
| **Session timeout** | Increase SESSION_TIMEOUT in config.php |
| **Images missing** | Use relative paths: `images/faculties/photo.jpg` |
| **JSON error** | Validate JSON: `php check_integrity.php` |
| **Can't access admin** | Try direct: `admin_login.php` URL |

## 📊 Faculty Data Structure

```json
{
  "name": "Dr. Full Name",
  "title": "Designation here",
  "image": "images/faculties/photo.jpeg"
}
```

**Fields:**
- `name` (required) - Full faculty name
- `title` (required) - Designation/position
- `image` (required) - Image file path

**Validation:**
- Name: 3-255 characters
- Title: 3-255 characters  
- Image: File must exist

## 📋 Emergency Protocol

**If data is lost or corrupted:**

1. **STOP** - Don't make more changes
2. **ASSESS** - Run `check_integrity.php`
3. **BACKUP** - Create new backup
4. **RESTORE** - Use `backup_manager.php`
5. **VERIFY** - Confirm data correct
6. **RESUME** - Continue operations
7. **LOG** - Document what happened

**Recovery time: 15-20 minutes**
