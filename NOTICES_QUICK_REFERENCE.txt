# NOTICES MANAGEMENT - QUICK REFERENCE

## 🚀 Quick Start

### Access
1. Login to Admin Panel: `admin@example.com` / `admin123`
2. Click "Notices" in navigation
3. URL: `/Project_SSV_Website/real/php/admin_notices.php`

## 📝 Add Notice (30 seconds)

```
Title:       "Event Name" (max 100 chars)
Description: "Details here" (max 500 chars)
Author:      "Optional - defaults to your name"
Date:        "27" (day number)
Month:       "Feb" (month abbr)
Publish:     "27-Feb-2026" (full date)

→ Click [Add Notice]
```

## ✏️ Edit Notice

1. Active Notices tab
2. Find notice → Click [Edit]
3. Modal opens with pre-filled data
4. Make changes
5. Click [Update Notice]

## 🗑️ Delete Notice

1. Active Notices tab
2. Click [Delete] for notice
3. Confirm deletion
4. Moves to Trash (recoverable)

## 🔄 Restore Notice

1. Trash tab
2. Click [Restore]
3. Returns to Active Notices

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| Changes not showing | Clear browser cache (Ctrl+Shift+Del) |
| "Failed to save" | Check file permissions (chmod 755) |
| Session expired | Login again |
| Character limit error | Use fewer characters |

## 💡 Pro Tips

1. **Author field**: Leave blank to auto-use your name
2. **Dates**: Use consistent format (DD-MMM-YYYY)
3. **Editing**: Use Edit button instead of Delete + Add
4. **Archive**: Don't permanently delete old notices
5. **Sync**: Website updates within 1 minute
